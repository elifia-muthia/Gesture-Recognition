# Gesture-Recognition
Gesture-Recognition Libary in C++

## OpenCV Setup
1. Install Homebrew
   
```/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"```

3. Install OpenCV
   
```brew install opencv```

5. Go to the project on VSCode and execute the following commands in the root

```mkdir build```

```cd build```

```cmake ..```

```make```

```./gesture-project```
